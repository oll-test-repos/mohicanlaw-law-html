Resolution No. 033-07 — Land Use Code

WHEREAS, The Stockbridge-Munsee Community, Band of Mohican Indians, is a federally recognized Indian tribe with inherent sovereign powers, and is governed by the Stockbridge-Munsee Tribal Council pursuant to the 1937 Constitution; and,

WHEREAS, The Stockbridge-Munsee Community has the authority under its constitution and its inherent rights as a sovereign nation to adopt and amend Ordinances; and,

WHEREAS, The Tribal Council adopted the Land Use Code on May 23, 2002, and amended the Code on May 19, 2004 and April 5, 2005; and,

WHEREAS, The Tribal Council desires to again amend the Land use Code by strengthening the enforcement section and better define terms including permit appeal process, the assisted family member clause, the jurisdictional clause.

NOW THEREFORE BE IT RESOLVED, that the Stockbridge-Munsee Tribal Council hereby adopts the following changes to Chapter 43, the Land Use Code by amending section §43.4; §43.5 (Accessory Use, Camping, Days, Junkyard, Land Use Review Board, Reservation, Uniform Dwelling Code); §43.7(A)(6)(a); §43.7(A)(6)(b); §43.7(A)(6)(d); §43.7(A)(6)(e); §43.7(D)(1)(g); §43.7(F); §43.8(3); §43.8(4); §43.10; §43.13(E); §43.15; §§43.16(A)(3)(b-e); §§43.16(B)(2-3); §§43.16(C)(1-2); §43.17(E); §43.18(A)(2)(b); §§43.18(C)(1-3); §43.19(A); §§43.19(C-D); §§43.20(A)(1-4); 43.20(B)(1); §43.21(A); §43.21(D); and creating §43.6; §43.7(A)(5)(f); §§43.7(B)(3)(g-k); §§43.7(C)(3)(0), (s-v); §43.7(3)(d); §§43.7(J)(3)(f-h); §43.8(6); §43.13(E); §43.18(A)(2)(c); §43.18(A)(3); §43.21(E).
